Download your land registry files to this folder

Format selected should be JSON
Here is a link to all the files by department :
https://cadastre.data.gouv.fr/data/etalab-cadastre/latest/geojson/departements/

You can also do it by town, by going to this link, select departpment and then INSEE code of the town :
https://cadastre.data.gouv.fr/data/etalab-cadastre/latest/geojson/communes/

ALL THE FILES HAVE TO MENTION "parcelles"
for example : ""cadastre-24-parcelles.json.gz"
Here is the link :
https://cadastre.data.gouv.fr/data/etalab-cadastre/latest/geojson/departements/24/cadastre-24-parcelles.json.gz

Do not forget to unzip the folders
